## How to run

In the project directory, run:

### `npm install`

Installs all needed dependencies.

Then you can run:

### `npm audit fix`

Fixes vulnerabilities.

Then run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
